# GenAI Course Assignments

## Overview
This repository contains assignments for the **GenAI Course** by TuteDude.

## Repository Structure
All assignments are organized in their respective course folders with assignment files stored within them.

### Example Structure
```
Assignment 1/
├── data_structures_assignment.ipynb
```

## Organization Guidelines
- Each assignment has its own dedicated folder (e.g., `Assignment 1/`)
- All related files for an assignment are contained within its folder
